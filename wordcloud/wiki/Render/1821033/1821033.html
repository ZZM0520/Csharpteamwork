Project Description
Tool to generate word cloud images from a list of words and frequencies.

Based off the algorithm in https://github.com/amueller/word_cloud

Nuget package at https://www.nuget.org/packages/WordCloud

Install (nuget)

Install-Package WordCloud

Usage

using WordCloudGen = WordCloud.WordCloud;

var wc = new WordCloudGen(width, height);

wc.Draw(words, frequencies);
